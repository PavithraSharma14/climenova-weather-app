## âœ¨ Installation

1. **Clone the repository**

```bash
git clone https://github.com/PavithraSharma14/climenova-weather-app.git
cd climenova-weather-app
```

2. **Install dependencies**

```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

4. **API Key Setup**<br>
Replace YOUR_API_KEY in api.js (or config) with your OpenWeatherMap API key.
